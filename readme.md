\# 极简通讯录 - 前端项目

\## 项目介绍

本项目是第一次作业的前端部分，实现极简通讯录的页面展示和用户交互，支持联系人的新增、查询、修改、删除功能。



\## 技术栈

\- HTML5（页面结构）

\- CSS3（样式美化）

\- JavaScript（交互逻辑）



\## 目录结构

src/

├── index.html       # 主页面

├── css/

│   └── style.css    # 样式文件

└── js/

└── script.js    # 交互逻辑文件

\## 本地运行步骤

1\. 确保电脑已安装 Python；

2\. 打开终端，进入项目的 `src` 目录（命令：`cd 你的项目路径/src`）；

3\. 启动本地服务器：执行命令 `python -m http.server 8080`。

4\. 打开浏览器，输入地址 `http://127.0.0.1:8080`，即可访问页面。



\## 功能说明

1\. 联系人添加：在表单中输入姓名和电话，点击“添加联系人”按钮提交；

2\. 联系人列表：自动加载所有联系人，显示姓名、电话及操作按钮；

3\. 联系人修改：点击列表中的“修改”按钮，在弹窗中编辑信息后提交；

4\. 联系人删除：点击列表中的“删除”按钮，确认后删除对应联系人。



\## 注意事项

1\. 本地运行时，需确保后端服务已在 `http://127.0.0.1:5000` 启动（否则无法正常交互）；

2\. 请勿修改目录结构，否则可能导致样式或脚本加载失败；
